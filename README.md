* 使用语言为python，界面设计为pyqt5，使用数据库为mysql
# 使用教程
* 在infomation.env中改好自己对应的配置
* 先启动firstcreate,py来创建数据库
* 最后，启动 main.py 即可。


# 目前实现功能
* 可导入 docx 文档
* 拥有注册和登录功能
* 可以查询记录
* 可以查看自己文档库里面内容
* 可以删除和增加用户
* 可以删除和增加文档库里面的文章
* 实现查重率和查重简单报告显示
* 报告导出
* 普通用户和管理员登录
* 管理员和普通用户之间不同的功能
* 尚未与数据库进行交互
